# 학생 성장 아카이브

## 소개

**학생 성장 아카이브**는 수학적상상력 학원에서 학생들의 점수 기록을 쉽고 간편하게 관리하기 위한 수업용 웹앱입니다.

학부모가 학생의 점수와 성장 기록을 한눈에 확인할 수 있도록 제작하였으며, 학생별 점수 기록과 반별 평균 점수, 누적 기록을 확인할 수 있습니다.

본 프로젝트는 HTML 하나만으로 제작한 단일 웹앱이며, `index.html` 파일을 실행하면 사용할 수 있습니다.

---

## 주요 기능

### 1. 점수 기록
- 학생 이름으로 로그인
- 학생의 점수 기록 확인

### 2. 반별 평균 점수
- 같은 반 학생들의 평균 점수 확인
- 개인 점수와 함께 비교 가능

### 3. 누적 기록 확인
- 월별 점수 기록 조회
- 학생의 성장 과정을 누적하여 확인

---

## 사용 방법

1. `index.html` 파일을 다운로드합니다.
2. 웹 브라우저에서 `index.html`을 실행합니다.
3. 학생 이름으로 로그인합니다.
4. 점수 기록을 확인합니다.
5. 반별 평균 점수와 누적 기록을 확인합니다.

---

## 수업용 주의사항

- 본 프로젝트는 웹 개발 수업을 위한 예제입니다.
- 하나의 `index.html` 파일로 구성되어 있습니다.
- 실제 개인정보는 입력하지 않습니다.
- 샘플 데이터만 사용하여 테스트합니다.
- 데이터는 실제로 저장되지 않을 수 있습니다.
- 실제 서비스로 운영하려면 아래 기능이 추가로 필요합니다.
  - 로그인 및 회원관리
  - 데이터베이스 저장
  - 개인정보 보호
  - 사용자 권한 관리
  - 보안 기능 강화
